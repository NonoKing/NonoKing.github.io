从零开始搭建个人博客（一）

发表于 2020-10-15 更新于 2020-11-03 分类于 Hexo 阅读次数：
本文字数： 639字

背景

无意间发现GitHub可以提供一个免费的域名，就是个人用户名.github.io

而且使用Hexo框架来搭建个人博客，简单容易上手，所以准备使用Hexo

从本文开始，我将记录自己建站的全部过程，也相当于是一个保姆级别的建站教程

废话不多说，let’s go!

准备工作

操作系统

Windows 10 64位
这个其实不怎么影响

工具

Git for windows
当然使用自带的PowerShell也是可以的

但是记得要以管理员身份运行以免会有一些权限问题

nods.js

接下来，去node.js官网下载并安装
以支持npm指令

Github配置

账号

你需要准备一个GitHub的账号，没有的话，就去注册一个
其中username就是上述提到的 个人用户名

如果你发现访问官网的速度很慢，可以参考上一篇文章来提高访问速度，以及git clone和git push的速度

ssh设置


点击右上角头像，在下拉菜单中选择Settings


在左侧的菜单中选择SSH and GPG keys


然后再右侧选择New SSH key


在title栏填写该key的名称，用这个没什么影响，一般来区分不同的电脑
比如你想在自己的电脑上写博客，在实验室或者公司也想同步更新，那这里就可以写
GitBash From xxx’s PC windows 或者 Shell from company Linux 之类的，总之可以随意写
为了之后在删除的时候，区分是哪个key

在key这一栏需要填写用GitBash生成的 .ssh 文件中的内容
